## Cheat Sheet

| **Parammeter**                  | **Usage**                    |
| ------------------------------- | ---------------------------- |
| -v                              | Show details                 |
| -i                              | Set the interface to be used |
| -w                              | Saves in a file              |
| -r                              | Read a file                  |
| -n                              | Shows IP instead of name     |
| -e                              | Details ethernet             |
| -A                              | Shows in ASCII               |
| -X                              | Shows in hex                 |
| protocol                        | Filter for protocol          |
| src host ip/dst host ip/host ip | Filter for IP                |
| port port_number                | Filter for port              |
