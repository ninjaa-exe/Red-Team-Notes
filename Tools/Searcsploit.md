## How to use
`searchsploit <program>`

## Cheat Sheet

| **Parameter**                    | **Usage**                                  |
| -------------------------------- | ------------------------------------------ |
| -u                               | Update the exploits database               |
| --exclude="argument \| argument" | Excludes an argument in the search         |
| -e                               | Performs an exact search on what you typed |
| -m codeID                        | Copy the exploit to your computer          |
